### 說明:
index( ), append( ), insert( ), remove( ), sort( ) 用於列表的指令
<ul>
<li>收尋出List的內容
<ul>
<li>index( )
</ul>
<li>添加新的值給列表
<ul>
<li>append( ): 只能放置在端點
<li>insert( ): 可以指定新增位置
</ul>
<li>刪除列表中的值
<ul>
<li>remove( )
</ul>
<li>排列List內值的順序
<ul>
<li>sort( )
</ul>
